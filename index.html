Formatle

Welcome to the Formatle

Formatle is a word game in the style of Wordle, but instead of guessing a word, you are guessing a file format extension!
Use the keyboard below, or use your keyboard to enter a guess, and press enter to submit it!

There will be a new file format every day, so make sure to check back every day for a new challenge!
The file format will be a random file format that is in the world of technology, such as .txt, .jpg, .html, .py, and many more!
The file format can include any letters or numbers, and can be many different lengths!

Good luck, and have fun!
